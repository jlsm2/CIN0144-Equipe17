## Instruções de Execução

### Dataset
O dataset é carregado automaticamente através da biblioteca `ucimlrepo`. Portanto:
-  **Não** é necessário fazer download do arquivo CSV por fora.
-  O script se encarrega de baixar e formatar os dados.

### Passo a Passo
1. Abra o arquivo `.ipynb` no Google Colab.
2. Execute a célula de instalação de pacotes (que contém `!pip install ucimlrepo`).
3. Vá no menu superior em **Runtime** > **Run all**.
